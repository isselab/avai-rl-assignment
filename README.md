# AVAI RoboRacer RL Assignment

This repository contains the Jupyter notebook and supporting files for the RoboRacer Reinforcement Learning assignment in the AVAI course.

## Contents

- `roboracer_rl_assignment_v2.ipynb`  
  Main Jupyter notebook for the assignment.

- `RL_assignment`  
  Assignment text/instructions.

- `RL_pipeline.png`  
  Supporting figure for the reinforcement learning pipeline.

## Attribution and References

Parts of this assignment are adapted from existing RoboRacer/F1TENTH, ROS 2, Gazebo, and reinforcement learning material.

The material has been modified and prepared for the AVAI course and the RoboRacer setup used at the Chair of Software Engineering, Ruhr University Bochum.
